# Ядро Анализатора Качества Кода Студенческих Проектов

## Основные возможности

- **AST-анализ**: Анализ структуры кода без его выполнения .
- **Плагинная архитектура**: Легкое добавление новых проверок через наследование классов-чекеров.
- **Хранение в БД**: Результаты каждого запуска сохраняются в PostgreSQL для отслеживания прогресса студента.

## Технологический стек

- **Язык**: Python 3.11+
- **Парсер**: Модуль `ast`
- **БД**: PostgreSQL
- **Тестирование**: Pytest

- **Установка:** Шаги по развертыванию среды и базы данных.
- **Docker**: Развертывание в Docker.
- **Архитектура:** Описание внутренней структуры и используемых паттернов.
- **Реализованные правила:** Подробный перечень правил, которые система проверяет.
- **Руководство для разработчика:** Инструкции по модификации и расширению ядра.
